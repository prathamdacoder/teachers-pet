# 🍎 Teacher's Pet - AI Worksheet Generator

> Generate custom worksheets in 5 seconds using AI!

## 🚀 Features

- ✅ AI-powered worksheet generation
- ✅ Multiple subjects (Math, Science, English)
- ✅ Multiple question types (MCQ, Fill in the Blank)
- ✅ Share worksheets with students
- ✅ Students can solve online
- ✅ Auto-grading!

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, Flask
- **AI:** OpenAI API / HuggingFace

## 🚀 Quick Start

1. Clone the repo
2. Add your OpenAI API key in `.env`
3. Run `pip install -r requirements.txt`
4. Run `python app.py`
5. Open `index.html`
